tumeur {lycee}R Documentation

tumeur/données de type biologique

Description

Pour étudier l'évolution des tumeurs, on en greffe sur des rats. Les données donnent les surfaces de la tumeur aprés 25 jours de greffe. Exemple de loi normale.

Usage

data(tumeur)
lire(tumeur)

Format

Un tableau contenant 1000 observations et une variable quantitative.

surface
une variable numérique.

Examples

data(tumeur) 
lire(tumeur) 
histogramme(surface)
moyenne(surface)
mediane(surface)
ecarttype(surface)
