diagrammeenbarre {lycee}R Documentation

Fonction qui trace le diagramme en barre

Description

La variable prend différentes valeurs ou modalités et le diagramme en barre présente graphiquement les fréquences respectives d' observation .

Usage

diagrammeenbarre(d)

Arguments

d une suite de nombres ou de modalités.

Examples

d=pileface(100)
diagrammeenbarre(d)
