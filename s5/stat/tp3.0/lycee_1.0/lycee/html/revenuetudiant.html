revenuetudiant {lycee}R Documentation

revenueetudiant/données économiques

Description

Revenus du travail d'été en Euros de 50 élèves inscrits à temps plein au cégep. Exemple de distribution normale.

Usage

data(revenuetudiant)
lire(revenuetudiant)

Format

Un tableau contenant 50 observations sur une variable.

Revenu
une variable numérique.

Examples

data(revenuetudiant)
lire(revenuetudiant)
histogramme(Revenu)
boitemoustache(Revenu)
mediane(Revenu)
moyenne(Revenu)
ecarttype(Revenu)
