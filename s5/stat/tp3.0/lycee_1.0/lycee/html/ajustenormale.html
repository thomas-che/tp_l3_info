ajustenormale {lycee}R Documentation

Fonction qui trace l'histogramme et la densité de la loi normale.

Description

Aprés avoir tracé l'histogramme d'une variable quantitative, on superpose la densité de la distribution normale de même moyenne et de même variance. On peut ainsi juger si la variable a une distribution normale (on dit aussi Gaussienne.)

Usage

ajustenormale(d)

Arguments

d une variable quantitative.

Examples

data(avion)
lire(avion)
tauxannulation=annulation/reservation
histogramme(tauxannulation)
ajustenormale(tauxannulation)
data(QI)
lire(QI)
histogramme(Qintellectuel)
ajustenormale(Qintellectuel)
