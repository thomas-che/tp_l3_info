camembert {lycee}R Documentation

Fonction qui trace le camembert ou diagramme en secteurs.

Description

La variable prend différentes valeurs ou modalités et le camembert présente graphiquement les fréquences respectives d' observation .

Usage

camembert(d)

Arguments

d une suite de nombres ou de modalités.

Examples

d<-jeudedes(100)
camembert(d)
data(guichet)
lire(guichet)
camembert(queue)
