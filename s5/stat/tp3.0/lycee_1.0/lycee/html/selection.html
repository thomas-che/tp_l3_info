selection {lycee}R Documentation

Fonction qui sélectionne une partie d'une colonne.

Description

On sélectionne le sous tableau d'une variable x en gardant les individus pour lesquels la variable y prend une valeur fixe

Usage

selection(d1,d2,valeur)

Arguments

d1 une variable .
d2 une autre variable observée sur les mêmes individus que x.
valeur une valeur prise par la variable d2

Examples


data(bebe)
lire(bebe)
selection(long,sexe,"fille")
