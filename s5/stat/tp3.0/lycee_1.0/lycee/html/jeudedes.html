jeudedes {lycee}R Documentation

Fonction qui simule le lancer d'un dés régulier.

Description

La fonction jeudedes(n) simule les résultats de n lancers d'un dés régulier.

Usage

jeudedes(n)

Arguments

n est le nombre de lancers

Examples

d<-jeudedes(1000)
frequence(d)
diagrammeenbarre(d)
camembert(d)
boitemoustache(d)
mediane(d)
