surgeles/étude de marché

Description

Les données sont issues d'une enquête portant sur les dépenses mensuelles en Euros pour les produits surgelés chez les ménages doté d'un congélateur 3 étoiles. Un exemple de distribution normale.

Usage

data(surgeles)
lire(surgeles)

Format

Un tableau de 500 observations sur une variable.

ventes
une variable numérique.

Examples

data(surgeles)
lire(surgeles)
histogramme(ventes)
boitemoustache(ventes)
mediane(ventes)
moyenne(ventes)
ecarttype(ventes)
