population {lycee}R Documentation

Fonction qui construit une population de 1000000 d'individus ayant une opinion oui ou non relativement à un sujet quelconque.

Description

La fonction population(p) construit une population de 1000000 d'individus pour laquelle une proportion p est favorable à une proposition quelconque. On peut tester la validité d'un sondage, qui approxime p à partir d'un échantillon tiré dans la population.

Usage

population(p)

Arguments

p est un nombre compris entre 0 et 1.

Examples

pop<-population(0.2)
s<-sondage(1000,pop)
frequence(s)
diagrammeenbarre(s)
camembert(s)
