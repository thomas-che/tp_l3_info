kystediscrimin1 {lycee}R Documentation

Aide au diagnostique de kystes ovariens /type médical

Description

Le tableau donne le diagnostique et les variables d'aide au diagnostique pour des kystes ovariens.

Usage

data(kystediscrimin1)

Format

Un tableau de 192 observations et 4 variables.

diagnos
3 diagnostiques possible pour les kystes KFfonctionnel KOB organique bénin KOM
organique malin
disc1
première variable d'aide au diagnostique.
disc2
deuxième variable d'aide au diagnostique.
coul
vert pour les KF orange pour les KOB rouge pour les KOM

Examples

data(kystediscrimin1)
lire (kystediscrimin1)
nuagedepoints(disc1,disc2,coul)
