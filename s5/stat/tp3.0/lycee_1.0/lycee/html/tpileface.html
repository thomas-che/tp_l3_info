tpileface {lycee}R Documentation

Fonction qui simule le jeu de pile ou face avec une pièce truquée

Usage

tpileface(n, pface)

Arguments

n est le nombre de lancers
pface est un nombre compris entre 0 et 1 qui désigne la probabilité de faire face

Examples

d<-tpileface(100, 0.7)
diagrammeenbarre(d)
frequence(d)
