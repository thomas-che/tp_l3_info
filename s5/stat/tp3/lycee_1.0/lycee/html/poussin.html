poussin {lycee}R Documentation

poussin/élevage

Description

On étudie l'évolution journalière du poids des poussins, en fonction des différents régimes qu'on leur impose.

Usage

data(poussin)

Format

A data frame with 578 observations on the following 4 variables.

poidsg
poids en grammes.
agej
age en jour.
nOpoussin
numéro du poussin.
régime
type du régime.

Examples

data(poussin)
lire(poussin)
selection(poidsg,régime,1)
couleur=ifelse(régime=='1','red',ifelse(régime=='2','green','orange'))
nuagedepoints(agej,poidsg,couleur)
