Abitibi {lycee}R Documentation

Abitibi/données géochimiques

Description

Analyses géochimiques de sédiments dans la région d'Abitibi dans un but de prospection minière.

Usage

data(Abitibi)
lire(Abitibi)

Format

un tableau de 62 observations et 8 variables.

Al2O3
pourcentage d'aluminium
FeO
pourcentagea numeric vector
MgO
pourcentagea numeric vector
CaO
pourcentagea numeric vector
Na2O
pourcentagea numeric vector
K2O
pourcentagea numeric vector
TiO2
pourcentagea numeric vector
G
facteur indiquant la présence g ou l'absence ng de gisement sur le lieu du prélèvement.

Examples

data(Abitibi)
lire(Abitibi)
hist(Al2O3)
mean(FeO)
mediane(MgO)
boitesparalleles(CaO,G)
histogramme(selection(Al2O3,G,"g"))
