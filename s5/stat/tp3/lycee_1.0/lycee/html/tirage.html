tirage {lycee}R Documentation

Fonction qui tire aléatoirement un échantillon dans une urne.

Description

La fonction tirage prélève un échantillon de taille n dans une urne. Le but est d'estimer la composition de l'urne, sans regarder toutes les boules. Cette méthode est intéressante s'il y a un trés grand nombre de boules dans l'urne.

Usage

tirage(n,u)

Arguments

n est la taille de l'échantillon
u

{ urne dans laquelle on tire}

Examples

u<-urne(100000,200000,40000)
t<-tirage(1000,u)
frequence(t)
frequence(u)
