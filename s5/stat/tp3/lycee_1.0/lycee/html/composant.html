composant {lycee}R Documentation

composant/contrôle qualité.

Description

On étudie la durée de vie de composants électroniques pour connaître par exemple les probabilités de panne des appareils dans lesquels ils interviennent.

Usage

data(composant)
lire(composant)

Format

Un tableau contenant 1000 observations et 1 variable.

vie
une variable numérique

Examples

data(composant)
lire(composant)
histogramme(vie)
ajusteexponentielle(vie)
