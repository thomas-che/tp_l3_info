pileface {lycee}R Documentation

Fonction qui simule le jeu de pile ou face avec une pièce équilibrée.

Description

La fonction pileface(n) simule les résultats de n lancers d'une pièce équilibrée.

Usage

pileface(n)

Arguments

n est le nombre de lancers

Examples

d<-pileface(1000)
d
frequence(d)
diagrammeenbarre(d)
camembert(d)
