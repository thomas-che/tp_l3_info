liretableauexcell {lycee}R Documentation

Fonction qui lit un tableau de données format csv.

Description

Cette fonction permet de lire un tableau de données excell enregistré au format csv . Les observations sont séparées par des points virgules et chaque colonne a un titre. Pour lire un tableau excell, on doit d'abord l'enregistrer au format csv. Le tableau doit-être dans le répertoire courant. Attention, en excell les nombres décimaux sont écris avec des virgules. Dans R ce sont des points.

Usage

nomdufichier=liretableauexcell('nomdufichier.csv')

Arguments

d chemin absolu suivi du nom du tableau.csv.
