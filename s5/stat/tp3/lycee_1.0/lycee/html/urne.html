urne {lycee}R Documentation

Fonction qui construit une urne contenant des boules. Chacune peut-être bleue, noire ou verte.

Usage

urne(n1,n2,n3)

Arguments

n1 est un nombre entier.
n2 est un nombre entier.
n3 est un nombre entier.

Examples

u<-urne(100000,40000,500000)
t<-tirage(1000,u)
frequence(t)
diagrammeenbarre(t)
camembert(t)
