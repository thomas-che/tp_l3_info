soleil {lycee}R Documentation

soleil/données météorologiques.

Description

Nombre d'heures d'ensoleillement à Montréal en Octobre de 1942 à 1990. Un exemple de distribution normale.

Usage

data(soleil)
lire(soleil)

Format

Un tableau contenant 49 observations sur une variable.

Nh
une variable numérique.

Examples

data(soleil)
lire(soleil)
histogramme(Nh)
boitemoustache(Nh)
moyenne(Nh)
mediane(Nh)
ecarttype(Nh)
