boitemoustache {lycee}R Documentation

Fonction qui trace la boite à moustache

Description

On trace la boite à moustache d'une variable quantitative. La boite est une représentation graphique des quartiles. Les moustaches représentent les valeurs extrèmes. Pour une distribution normale, la boite est symétrique.

Usage

boitemoustache(d)

Arguments

d une variable quantitative.

Examples

d<-jeudedes(100)
boitemoustache(d)
data(bebe)
lire(bebe)
boitemoustache(long)
mediane(long)
