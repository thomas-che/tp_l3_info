avion {lycee}R Documentation

Dossier renseigné de 2000 vols dans une compagnie aérienne/gestion

Description

Une compagnie aérienne souhaite faire une étude statistique pour connaître le comportement de ses passagers afin d' ajuster au mieux sa politique de surréservation. Pour cela, elle relève pour 2000 vols, le nombre de réservations, le nombre d’ annulation, le type de vol (vacances ou professionnel), le prix du billet.

Usage

data(avion)

Format

Un tableau de 2000 lignes et 5colonnes.

vol
{numéro du vol}
reservation
le nombre de réservation
annulation
le nombre d'annulation
butvoyage
le but du voyage, professionnel ou vacances
prix
le prix du billet

Examples

data(avion)
lire(avion)
tauxannulation=annulation/reservation
selection(tauxannulation,butvoyage,"vacances")
boitesparalleles(tauxannulation,butvoyage)
