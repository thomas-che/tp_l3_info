boitesparalleles {lycee}R Documentation

Fonction qui trace sur le même graphique différentes boites à moustaches .

Description

On trace les boites à moustaches respectives d'une variable quantitative, pour les différentes modalités d'une variable qualitative donnée. Par exemple, si on trace à côte à côte les boites à moustache des poids des bébés garçons et filles, on voit que la distribution du poids est différente selon le sexe et qu'il faut séparer les bébés garçons et les bébés filles pour étudier le poids.

Usage

boitesparalleles(d,f)

Arguments

d une variable quantitative.
f

{ une variable qualitative qui partitionne la population.}

Examples


data(bebe)
lire(bebe)
boitesparalleles(poids,sexe)
