sondage {lycee}R Documentation

Fonction qui prélève aléatoirement un échantillon dans une population.

Usage

sondage(n,pop)

Arguments

n est la taille de l'échantillon
pop

{ population dans laquelle on tire l'échantillon }

Examples

pop<-population(0.7)
s<-sondage(100,pop)
frequence(s)
frequence(sondage(100,tpileface(10000,0.6)))
