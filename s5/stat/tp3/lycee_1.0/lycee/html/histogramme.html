histogramme {lycee}R Documentation

Fonction qui trace l'histogramme

Description

On trace l'histogramme d'une variable quantitative. C'est une représentation de la distribution de la variable. Pour chaque classe, l'aire du rectangle sousl'histogramme donne une valeur proportionnelle à la fréquence des données qui se trouvent dans la classe. Pour une distribution normale, l'histogramme a une forme de cloche.

Usage

histogramme(d)

Arguments

d une variable quantitative.

Examples


data(bebe)
lire(bebe)
moyenne(long)
histogramme(long)
ajustenormale(long)
