regression {lycee}R Documentation

Fonction qui calcul la droite de régression d'un nuage de points et trace le graphique.

Description

On calcule l'équation de la droite de régression d'un nuage de points. On fait la représentation graphique.

Usage

regression(x,y)

Arguments

x une variable quantitative explicative.
y une variable quantitative à expliquer.

Examples

data(bebe)
lire(bebe)
regression(long,poids)
