ajusteexponentielle {lycee}R Documentation

Fonction qui trace l'histogramme et la densité de la loi exponentielle.

Description

Aprés avoir tracé l'histogramme d'une variable quantitative, on superpose la densité de la loi exponentielle de paramètre 1/moyenne. On peut ainsi juger si la variable suit une loi exponentielle.

Usage

ajusteexponentielle(d)

Arguments

d une variable quantitative.

Examples

data(robinet)
lire(robinet)
histogramme(consommation)
ajusteexponentielle(consommation)
